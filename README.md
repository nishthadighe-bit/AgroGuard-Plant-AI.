# 🌿 AgroGuard: Intelligent Plant Pathology System

An AI-powered web application designed to detect 38 different plant diseases from leaf images with high confidence. Built for farmers and researchers to get instant diagnostics with multi-modal voice feedback.

## 🚀 Features
* **Deep Learning Engine:** Uses a trained CNN model (MobileNetV2) with 95%+ training accuracy.
* **Multi-Modal Feedback:** Not only shows the result but **speaks** the diagnosis using Text-to-Speech (gTTS).
* **Instant Deployment:** Hosted on Streamlit Cloud for easy mobile and desktop access.

## 🛠️ Tech Stack
* **Language:** Python
* **ML Framework:** TensorFlow / Keras
* **Web Interface:** Streamlit
* **Voice Synthesis:** gTTS (Google Text-to-Speech)
* **Dataset:** PlantVillage (54,000+ images)

## 📖 How to Use
1. Upload a clear image of a single plant leaf.
2. Wait for the AI to analyze the patterns.
3. View the diagnosis and listen to the automated voice report.

## 🧪 Results
During testing, the model achieved a real-world validation confidence of **~87%** on unseen potato leaf samples.
